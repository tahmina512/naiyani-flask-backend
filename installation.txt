   Installation instructions: 
    -------------------------------------
    1. clone the project from GitHub repository: git clone <repository_url>
    =>git clone https://github.com/tahmina512/Naiyani-replicated.git

    2. open the project in vs code

    3. Install Python (version 3.10.2 (pip 23.3.1) or updated one) 
    doc: https://www.python.org/downloads/macos/ (installation doc with a different version)

    4. if pip is not upgraded type in the cmd line: 
    => pip install --upgrade --user pip  

    5. install pandas library 
     =>pip install pandas

    6.install flask and flask-cors: 

    => pip install Flask Flask-CORS

   7. run the project :
   => python app.py
 
   follow this document: https://flask.palletsprojects.com/en/3.0.x/installation/
